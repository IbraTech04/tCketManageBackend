# tCketManage Documentation

The following directory contains various markdown files that document various aspects of tCketManage. Expect documentation to lag behind development in the early stages, as we are simply moving too fast to keep up with writing docs, but we will get there eventually. For now, the best way to understand how to use the package is to read through the code and the tests, which are fairly well commented.
